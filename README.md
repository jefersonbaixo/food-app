# food-app

This app is made to work with [Yelp API](https://www.yelp.com/developers/documentation/v3).
You Need an API Key to run the App.

## Installation

Just run

```bash
yarn install
```

or

```bash
npm install
```

## Configuration

You have to rename the file :
```bash
src/config.example.env.js
```
to
```bash
src/config.env.js
```
and paste your API key there.

## Running
```bash
yarn start
```
or
```bash
npm start
```
